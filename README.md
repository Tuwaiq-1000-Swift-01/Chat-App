# Chat-App
Create chat App,  Using Firebase 

## Features
- Email/Pass Registration / Log In
- Real Time Conversations
- Search for Users
- User Profile
- Dark Mode Support

## TODO
- [ ] Photo Messages

## Set Up

After cloning the repo, you'll need to run `pod install` to install cocoapod dependencies.
